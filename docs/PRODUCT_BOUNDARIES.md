# 产品与研究边界记录

本文记录 2026-06-24 会议后确定的近期产品边界，避免把远期肌肉骨骼实时孪生误并入当前 Stage 2 切口 Agent 目标。

## 当前聚焦

- 表皮 RSTL / 皮纹线迁移：照片、视频、摄像头和标准脸/3D 标注轨。
- 病灶与肿物处理：医生输入中心、直径、深度、切缘、椭圆/自由轮廓。
- 候选切口设计：皮下线性切口、皮表梭形切口、敏感结构 guardrails、医生审阅导出。
- 低置信度辅助线索：自然皱襞、皱纹、皮表肿物边界候选，只作为 secondary cue。
- 临床验证闭环：指标、失败分类、审阅记录、隐私与审计边界。

## 当前暂缓

肌肉骨骼实时孪生、术中级软组织/肌肉骨骼耦合模拟、实时生物力学手术规划不属于当前阶段目标。原因：

- 数据需求、算力、合规和临床验证复杂度明显高于当前表皮 RSTL 与病灶处理。
- 近期切口 Agent 的安全边界是“决策辅助可视化 + 医生复核”，不是自主术中规划。
- 当前演示中的“切除闭合演示”只保留沿 RSTL 的定性表面张力直觉，不扩展成临床级有限元或肌肉骨骼模拟。

## 会议结论同步记录

- 删除切除闭合演示中的红色逆 RSTL 对比线，只保留沿 RSTL 的绿色演示路径。
- 暂缓肌肉骨骼实时孪生；若未来重启，另开决策 gate，评估数据、算力、临床意义、合规风险和验证指标。
- #64 Stage 2 agentic 切口规划只覆盖表皮 RSTL、病灶表达、候选切口、guardrails 和医生审阅，不与肌肉骨骼实时孪生混淆。

## 未来重启条件

至少需要先满足：

- 已有受控真实病例数据和伦理/合规审批。
- 已定义可复现的软组织/肌肉骨骼验证指标。
- 已明确部署形态、算力预算、访问控制和日志/审计策略。
- 已由临床团队确认该方向的近期研究价值高于继续完善表皮 RSTL 与病灶闭环。
